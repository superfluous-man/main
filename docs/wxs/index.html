    本筆記主要基於以下三本書整理：

    • 袁行霈. 中国文学史（四卷本）[M]. 北京:高等敎育出版社, 2013

    • 王國瓔. 中國文學史新講（兩卷本）[M]. 臺北: 聯經出版公司, 2014

    • 章培恒, 骆玉明. 中国文学史新著（三卷本）[M]. 上海: 复旦大学出版社, 2014

    袁書是流行最廣泛的課本，行文四平八穩，觀點中規中矩，文獻工夫扎實，是十分優秀的「課本」。本書編輯團隊頗爲龐雜，各章文筆偶見特色，未盡統一。筆者當年從魯小俊老師學習文學史課程，即以此書爲課本。

    王書編寫，文體爲經，時代爲緯，觀點中規中矩，鮮有突破，是合格的「參考課本」。本書作者據說是家學淵源深厚的女性，全書文筆較爲清新明快。

    章書乍一讀，頗覺驚豔，仔細體悟，則似有諸多缺陷：全書似乎是先有「人性是發展的」總論點基調，再去找材料套進去這條一以貫之的總線索，邏輯看起來一貫明了，論證卻時常缺乏說服力。因邏輯自足，全書文風顯得酣暢大氣，某些段落有排山倒海之勢，因而頗見作者性情；也因此導致寫作時語言表述方面瑕疵不少。讀者容易掉進作者的「邏輯圈套」而走火入魔，本書應是文學史學習的「另類參考課本」。
